# Gestion de Réservation Hôtel

## Description
Application web de gestion de réservation d'hôtel développée avec Flask, HTML et CSS.

Le système permet aux clients de créer un compte, de se connecter et de réserver une chambre selon une date choisie.  
L'administrateur peut se connecter avec un compte admin afin de consulter et gérer les réservations des clients.

## Fonctionnalités

### Côté Client
- Création d’un compte client
- Connexion client avec email et mot de passe
- Choix d’une chambre
- Choix d’une date de réservation
- Vérification de la disponibilité de la chambre

### Côté Admin
- Connexion avec un compte administrateur
- Consultation de toutes les réservations
- Suppression d’une réservation
- Gestion des réservations clients

## Technologies utilisées
- Python
- Flask
- HTML
- CSS
